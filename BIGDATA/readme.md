Primo progetto Big Data
----------------------------------------------------------

Questo progetto diviso per punti, contiene alcuni esercizi svolti durante il corso di Big Data dell'università ROMA TRE.
L'esercitazione prevedeva la realizzazione di diversi Job in Java su input di grandezza crescente. I diversi Job sono stati implementati tramite:
1) Map Reduce
2) Pig
3) Hive

Inoltre l'esercitazione prevedeva:
4) Utilizzo di Hadoop in locale e su Server Amazon
5) Creazione di Log di esecuzione 
6) Realizzazione in Pseudocodice dei Job
7) Tabelle e grafico tempi di esecuzione
